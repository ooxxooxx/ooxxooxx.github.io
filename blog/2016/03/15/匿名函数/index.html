匿名函数

  • Scala的匿名函数,形如：
scala> (x:Int) => x+1
res1: Int => Unit = <function1>

其中 => 左边是字面量的参数列表，=>右边是函数体。

匿名函数是有类型的,上式类型为： Int => Int

  • 匿名函数可以包含多行表达式，要用｛｝括起来
scala> (x:Int) => { println(x)
     | println(x + 1)
     | }
res2: Int => Int = <function1>
  • 匿名函数可以赋给一个val变量
scala> val f =(x:Int) => x+1
f: Int => Int = <function1>
  • 匿名函数可以作为参数传递
scala> def foo(func : Int => Unit){ func(0) }
res3: f: Int => Int = <function1>

这里，foo的参数func的类型是Int => Unit,即参数为一个Int类型，返回值为Unit类型的函数。 另外还有，单参数、多参数、无参数三种类型

关于函数字面量和lambda表达式

一些文章中，会出现函数字面量和lambda表达式。
个人认为，这两个概念与匿名函数说的是同一个语义概念。
因为在Scala的文档中，只出现了匿名函数,并且参考了一些文章，也都没有特别指明函数字面量或lambda表达式有什么特殊的地方。暂且这样理解，概念上就不深究了。
